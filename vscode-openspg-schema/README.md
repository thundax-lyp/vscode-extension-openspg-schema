[OpenSPG Schema](https://github.com/OpenSPG/KAG) is the mark language used in OpenSPG server to create knowledge graph structure, this extension provides:

* Syntax highlighting
* Document symbols
* Definition and references
* Hover information
* Formatting
* Diagnostics
* Schema preview
